# Adire Maison

A premium fashion designer website in Vue 3 featuring West African native couture.

## Setup Instructions

1. `npm install`
2. `npm run dev`        → local dev at localhost:5173
3. `npm run build`      → production build to /dist

## Vercel Deployment

1. Push to GitHub
2. Go to vercel.com → Add New Project → Import GitHub repo
3. Vercel auto-detects Vite → Framework: Vite → Build: `npm run build` → Output: `dist`
4. Click Deploy ✅
